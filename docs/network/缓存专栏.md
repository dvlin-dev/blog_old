# 前端到底有哪些缓存？

## 按照失效策略划分可以分为：

- 强缓存
- 协商缓存

## 浏览器缓存按照缓存位置划分可以分为：

- Service Worker Cache
- Memory Cache
- Disk Cache
- Push Cache

##  存储型缓存 
Cookie、Web Storage、IndexedDB 等

## other
